# Novolis.Physics (meta-package)

Installs the full Novolis physics stack in one reference:

```bash
dotnet add package Novolis.Physics
```

## Documentation

- [Repository README](https://github.com/frankhaugen/novolis-physics/blob/main/README.md) — quick start and conventions
- [Integration guide](https://github.com/frankhaugen/novolis-physics/blob/main/docs/INTEGRATION.md) — which API to use
- [Architecture](https://github.com/frankhaugen/novolis-physics/blob/main/docs/ARCHITECTURE.md) — packages and integration styles
- [Examples](https://github.com/frankhaugen/novolis-physics/tree/main/docs/examples) — ballistics, collision, DI

Or reference individual packages (`Novolis.Physics.Motion`, `Novolis.Physics.Orbits`, …) if you only need a subset.
